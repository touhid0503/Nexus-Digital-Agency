# Nexus Digital Agency

A professional, responsive one-page landing page built using modern web technologies.

## 🚀 Technologies Used
- **HTML5**: Semantic structure.
- **Tailwind CSS**: Modern utility-first styling and responsiveness.
- **Custom CSS**: Brand-specific animations and UI enhancements.
- **Google Fonts**: Inter font family for modern typography.

## ✨ Features
- Fully Responsive (Mobile, Tablet, Desktop).
- Sticky Navigation with backdrop-blur effect.
- Smooth scrolling navigation.
- Floating Hero animation.
- Portfolio hover effects.
- Clean and modern "Slate & Blue" UI.
